# InternGuardAPI

InternGuardAPI is a web application designed to identify potentially fraudulent internship postings and improve student awareness of online internship scams.

## Features

- Detects suspicious internship listings
- Flags common scam indicators
- Responsive user interface
- Easy deployment using Docker
- Simple and lightweight architecture

## Technologies Used

### Frontend
- HTML5
- CSS

### Backend
- Python
- Flask

### DevOps
- Docker
- Git
- GitHub
